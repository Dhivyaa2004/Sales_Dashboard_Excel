# Sales Performance Dashboard in Excel

## Overview
An interactive sales dashboard built using Microsoft Excel to analyze sales performance and business trends.

## Objectives
- Monitor overall sales performance.
- Analyze profit and revenue trends.
- Compare product category performance.
- Evaluate salesperson contributions.
- Understand country-wise sales distribution.
- Support business decision-making through data visualization.

## Features
- Sales Analysis
- Profit Analysis
- Product-wise Performance
- Country-wise Sales
- Salesperson Performance
- Pivot Tables
- Interactive Charts

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
